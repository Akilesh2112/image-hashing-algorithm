# Hash-js [![npm](https://img.shields.io/npm/dt/opnl?label=OPNL%20Main)](https://www.npmjs.com/package/opnl)
### Hexa-decimal image hashing

# Application setup:
## Clone the repo 
```git clone https://github.com/wee-5/Hash-js.git```

## Install the dependencies
```npm install```

## Run the application
```node index.js```
